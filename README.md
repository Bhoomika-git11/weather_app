# 🌦️ Weather Forecasting App — India

A real-time desktop Weather App built with **Python & Tkinter** using the **OpenWeatherMap API**.

## ✨ Features
- 🕐 Live clock & date
- 🌡️ Temperature & Feels Like
- 💧 Humidity, Pressure, Wind & Visibility
- 🌅 Sunrise & Sunset timings
- 🌌 Animated purple galaxy UI
- 📍 All Indian states & cities supported

## 🛠️ Setup
```bash
pip install requests
python weather_app.py
```

## 🔑 API Key
Get a free key from [openweathermap.org](https://openweathermap.org) and replace in `weather_app.py`:
```python
API_KEY = "your_api_key_here"
```
